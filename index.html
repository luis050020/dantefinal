Submódulo 1 - Construye aplicaciones web

Estructura HTML Básica

HTML es el lenguaje fundamental para crear páginas web. Cada elemento HTML tiene un propósito específico en la estructura del documento.

HTML 

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>¡Bienvenidos a HTML!</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>
</html>

Explicación:

Esta es la estructura básica de un documento HTML5. El DOCTYPE declara el tipo de documento, el elemento html contiene todo el contenido, head incluye metadatos y configuraciones, y body contiene el contenido visible.

Estilos con CSS

CSS permite dar estilo y diseño a nuestras páginas web, controlando colores, tamaños, disposición y más.

CSS 

/* Estilos básicos */
body {
  font-family: Arial, sans-serif;
  line-height: 1.6;
  color: #333;
}

/* Diseño de caja */
.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 0 20px;
}

/* Flexbox */
.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

/* Grid */
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

/* Responsive design */
@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}

Explicación:

Este ejemplo muestra varios conceptos de CSS: estilos básicos, diseño de caja, Flexbox, Grid y diseño responsivo. Cada sección demuestra cómo controlar diferentes aspectos del diseño y la presentación de una página web.

JavaScript Interactivo

JavaScript permite crear interactividad en las páginas web, respondiendo a eventos y manipulando el contenido dinámicamente.

JavaScript 

// Variables y funciones
let contador = 0;
const incrementar = () => {
  contador++;
  actualizarContador();
};

// Manipulación del DOM
function actualizarContador() {
  document.getElementById('contador').textContent = contador;
}

// Eventos
document.getElementById('boton-incrementar').addEventListener('click', incrementar);

// Promesas y fetch
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

// Async/Await
async function obtenerDatos() {
  try {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error:', error);
  }
}

Explicación:

Este ejemplo muestra varios conceptos de JavaScript: variables, funciones, manipulación del DOM, eventos, promesas, fetch y async/await. Cada sección demuestra cómo JavaScript puede interactuar con la página web y realizar operaciones asíncronas.

Submódulo 2 - Implementa aplicaciones web

En este submódulo, aprenderás cómo implementar las herramientas y conceptos de desarrollo web en proyectos del mundo real.

Caso de Estudio: Creación de un Portafolio en Línea

Vamos a ver cómo podemos aplicar HTML, CSS y JavaScript para crear un portafolio en línea profesional.

HTML 

<!-- Estructura básica del portafolio -->
<header>
    <h1>Mi Portafolio</h1>
    <nav>
        <ul>
            <li><a href="#sobre-mi">Sobre Mí</a></li>
            <li><a href="#proyectos">Proyectos</a></li>
            <li><a href="#contacto">Contacto</a></li>
        </ul>
    </nav>
</header>

<main>
    <section id="sobre-mi">
        <h2>Sobre Mí</h2>
        <p>Soy un desarrollador web apasionado por crear experiencias digitales únicas.</p>
    </section>

    <section id="proyectos">
        <h2>Mis Proyectos</h2>
        <div class="proyecto">
            <h3>Proyecto 1</h3>
            <img src="proyecto1.jpg" alt="Proyecto 1">
            <p>Descripción del proyecto 1</p>
        </div>
        <!-- Más proyectos aquí -->
    </section>

    <section id="contacto">
        <h2>Contacto</h2>
        <form id="formulario-contacto">
            <input type="email" placeholder="Tu email" required>
            <textarea placeholder="Tu mensaje" required></textarea>
            <button type="submit">Enviar</button>
        </form>
    </section>
</main>

<footer>
    <p>&copy; 2023 Mi Portafolio. Todos los derechos reservados.</p>
</footer>

Este es un ejemplo básico de cómo estructurar un portafolio en línea. En un proyecto real, añadirías más contenido y secciones según tus necesidades específicas.

Implementación en el Mundo Real

Para implementar este portafolio en un entorno de trabajo real, considerarías los siguientes pasos:

  1. Diseño y planificación: Crear wireframes y mockups antes de comenzar a codificar.
  2. Desarrollo: Escribir el código HTML, CSS y JavaScript, siguiendo las mejores prácticas y estándares de la industria.
  3. Pruebas: Realizar pruebas exhaustivas en diferentes navegadores y dispositivos.
  4. Despliegue: Subir el sitio a un servidor web y configurar un dominio.
  5. Mantenimiento: Actualizar regularmente el contenido y las tecnologías utilizadas.
