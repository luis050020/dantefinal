Estructura HTML Básica

HTML es el lenguaje fundamental para crear páginas web. Aquí encontrarás ejemplos prácticos.

Estructura Básica 

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página HTML</title>
</head>
<body>
    <h1>¡Hola, mundo!</h1>
    <p>Este es un párrafo en HTML.</p>
</body>
</html>

Explicación:

Esta estructura básica de HTML define el esqueleto de cualquier página web. Incluye el DOCTYPE, metadatos y el contenido principal dentro del elemento ``.

Formularios 

<form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required><br>
    <label for="edad">Edad:</label>
    <input type="number" id="edad" name="edad"><br>
    <label for="mensaje">Mensaje:</label><br>
    <textarea id="mensaje" name="mensaje" rows="4"></textarea><br>
    <button type="submit">Enviar</button>
</form>

Explicación:

Este formulario incluye etiquetas para campos de texto, números y un área de texto para mensajes. Cada campo está relacionado con un `

Tablas 

<table border="1">
    <thead>
        <tr>
            <th>Nombre</th>
            <th>Edad</th>
            <th>Ciudad</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Juan</td>
            <td>25</td>
            <td>Madrid</td>
        </tr>
        <tr>
            <td>María</td>
            <td>30</td>
            <td>Barcelona</td>
        </tr>
    </tbody>
</table>

Explicación:

Las tablas se utilizan para mostrar datos en un formato estructurado, con encabezados en `` y filas en ``.

Contenido Multimedia 

<figure>
    <img src="imagen.jpg" alt="Descripción de la imagen">
    <figcaption>Pie de foto explicativo.</figcaption>
</figure>

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Tu navegador no soporta audio.
</audio>

<video controls>
    <source src="video.mp4" type="video/mp4">
    Tu navegador no soporta video.
</video>

Explicación:

Este ejemplo incluye elementos multimedia como imágenes con pie de foto, audio y video con controles integrados.

Estilos con CSS

CSS permite dar estilo y diseño a nuestras páginas web. Aquí se incluyen ejemplos clave:

Estilos Básicos 

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    color: #333;
}
h1 {
    color: #007BFF;
    text-align: center;
}

Explicación:

Define estilos generales para texto y encabezados, aplicando fuentes y colores personalizados.

Flexbox 

.container {
    display: flex;
    justify-content: space-around;
    align-items: center;
    height: 100vh;
}
.item {
    background: #007BFF;
    color: #fff;
    padding: 20px;
    border-radius: 8px;
}

Explicación:

Flexbox es un modelo de diseño poderoso que permite distribuir elementos de forma flexible y alinearlos vertical y horizontalmente.

Grid 

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}
.card {
    background: #f4f4f4;
    padding: 20px;
    border: 1px solid #ccc;
}

Explicación:

Grid Layout permite crear diseños complejos dividiendo el espacio en áreas cuadradas organizadas.

Animaciones 

@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}
.button {
    animation: fadeIn 2s ease-in-out;
}

Explicación:

Las animaciones CSS permiten agregar transiciones visuales dinámicas, como este efecto de "aparecer".

JavaScript Interactivo

JavaScript agrega interactividad a nuestras páginas web. Aquí se muestran varios ejemplos:

Contador 

let contador = 0;
const incrementar = () => {
    contador++;
    document.getElementById('contador').textContent = contador;
};
document.getElementById('increment-btn').addEventListener('click', incrementar);

Explicación:

Este ejemplo muestra cómo actualizar el DOM en tiempo real respondiendo a un evento de clic.

Fetch API 

fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Explicación:

Fetch API permite realizar solicitudes HTTP y manejar datos de manera asíncrona.

Validación 

const formulario = document.getElementById('miFormulario');
formulario.addEventListener('submit', function(event) {
    event.preventDefault();
    const nombre = formulario.nombre.value;
    if (nombre === '') {
        alert('El campo Nombre es obligatorio');
    } else {
        alert('Formulario enviado');
    }
});

Explicación:

Este código evita el envío de un formulario vacío, mostrando una alerta si el campo requerido está vacío.

DOM 

const boton = document.getElementById('cambiarColor');
boton.addEventListener('click', function() {
    document.body.style.backgroundColor = '#f0f0f0';
});

Explicación:

Manipula el DOM para cambiar dinámicamente el color de fondo al hacer clic en un botón.
